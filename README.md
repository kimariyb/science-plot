## 项目简介

Kimariyb 的科研绘图脚本

- `line curve` 用于绘制 1D 的各类曲线图的脚本
    - `fraction` 用于绘制分析化学分布分数曲线图的脚本
    - `NICS-1D` 用于绘制 NICS 曲线图的脚本
    - `scan-relax` 用于绘制柔性或刚性扫描的能量曲线图的脚本
    - `uv-vis` 用于绘制 UV-Vis 多曲线图脚本
    - `irc` 用于绘制 irc 多子图脚本
